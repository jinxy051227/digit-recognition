# z
# Digit Recognition Project

## Overview
This project uses machine learning to classify handwritten digits (0–9).

## Skills
- Python
- scikit-learn
- Machine Learning
- Data Analysis

## Description
This project trains a model using the sklearn digits dataset.
The model can recognize handwritten numbers.

## Result
The model achieved good accuracy in classifying digits.

## How to Run
1. Install Python
2. Install required libraries:
   pip install scikit-learn
3. Run the script:
   python main.py
